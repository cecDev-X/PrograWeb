# Ejercicios de JavaScript

Se aplican lo aprendido hasta el momento de Programacion Web
Son 10 ejercicios utilizando el stack basico minimo para paginas web, entre ellos se incluye:
* **Conversor de Temperatura:** Una calculadora sencilla que convierte grados Celsius a Fahrenheit utilizando validaciones de campos vacíos y protección de datos mediante atributos `readonly`.

## Se utilizó:
* **HTML5:** Estructura semántica de los formularios y el menú principal.
* **CSS3:** Estilos limpios, diseño responsivo y variables de color para mantener consistencia visual.
* **JavaScript :** Lógica de programación, eventos del lado del cliente y validación de datos.
